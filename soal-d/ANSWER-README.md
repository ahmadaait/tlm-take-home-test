# Assessment Point D

## Installation Requirements

1. PHP
2. Node.js
3. NPM or Yarn (optional)

### PHP

1. Go to `php` directory.
2. Run `index.php` file.

   ```bash
   php index.php
   ```

### Node.js

1. Go to `js` directory.
2. Install dependencies.

   ```bash
   yarn install
   ```

3. Run `test.js` file.

   ```bash
   node test.js
   ```
